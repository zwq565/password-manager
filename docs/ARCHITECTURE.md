## 核心特性

1. **端到端加密** - 所有数据在客户端加密
2. **零知识架构** - 服务器无法访问明文
3. **全球分布** - 基于 Cloudflare 边缘网络
4. **高性能** - Workers 边缘计算，毫秒级响应
5. **安全认证** - JWT + HTTPS + CORS
