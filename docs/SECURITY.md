# 安全政策

## 加密标准

- **对称加密**: AES-256-GCM
- **密钥派生**: PBKDF2 (100,000 iterations)
- **哈希**: SHA-256
- **随机数生成**: libsodium.js

## 安全架构

```
主密码 → PBKDF2 → 加密密钥 → AES-256-GCM → 密文
```

## 零知识架构

✅ **服务器永远看不到真实密码**
✅ **所有加密都在客户端完成**
✅ **密钥永不上传到服务器**

## 防御威胁

| 威胁 | 防御方式 | 有效性 |
|------|---------|--------|
| 中间人攻击 | HTTPS + HSTS | ✅ 完全 |
| 数据库泄露 | 端到端加密 | ✅ 完全 |
| 服务器入侵 | 零知识架构 | ✅ 完全 |
| 暴力破解 | 速率限制 | ✅ 高度 |

## 密码策略

- 最少 12 字符
- 大小写字母 + 数字 + 特殊字符
- 避免常见密码

## 审计日志

所有操作都被记录:
- 登录/注册
- 密码创建/修改/删除
- 异常活动
